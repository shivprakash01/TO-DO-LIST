# 📝 To-Do List Application

A simple and efficient **To-Do List web application** that helps users organize and manage daily tasks. Users can add, complete, and delete tasks to stay productive and track their progress.

---

## 🚀 Live Demo

🔗 **Try the application here:**
https://shivprakash01.github.io/TO-DO-LIST/

---

## 📌 Features

* ➕ Add new tasks
* ✅ Mark tasks as completed
* 🗑️ Delete tasks
* 📋 Simple and clean user interface
* ⚡ Fast and lightweight frontend application

---

## 🛠️ Built With

* **HTML5** – Structure of the application
* **CSS3** – Styling and layout
* **JavaScript (Vanilla JS)** – Application functionality

---

## 📂 Project Structure

```
TO-DO-LIST
│
├── index.html   # Main HTML structure
├── index.css    # Styling for the application
└── app.js       # JavaScript logic for task management
```

---

## 💡 How to Use

1. Open the application in your browser.
2. Type a task in the input field.
3. Click **Add** to add the task to the list.
4. Mark tasks as completed or delete them when finished.
